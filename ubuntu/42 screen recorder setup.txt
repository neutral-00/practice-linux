## OBS Studio or Open Broadcaster Software Studio
- I selected OBS because it is feature rich. and it is supported in all 3 OS
- install:
sudo add-apt-repository ppa:obsproject/obs-studio
sudo apt update
sudo apt install ffmpeg obs-studio

### launch it from the terminal
$ obs
